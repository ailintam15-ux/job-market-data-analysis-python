# Job Market Data Analysis using Python

This project explores job market trends using Python to understand demand for different job roles and salary distribution.

## Tools Used
* Python
* Pandas
* Matplotlib
* Seaborn

## Dataset
Job market dataset containing job titles and salary information.

## Key Analysis
* Identify the most in-demand job roles
* Explore salary distribution
* Perform exploratory data analysis (EDA)

## Insights

* Data-related roles such as Data Analyst and Data Scientist appear frequently in the dataset.
* Salary distribution shows variation depending on role and experience level.
* Technical roles dominate higher salary ranges.

These insights demonstrate how exploratory data analysis can reveal patterns in job market demand.
